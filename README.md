# OpticsShop — костяк інтернет-магазину

Демо-скелет каталогу (сторінка категорії «Тепловізори»), натхненний структурою [optics-pro.com.ua](https://www.optics-pro.com.ua/ua/teplovizori/).

**Це не клон і не афілійований сайт.** Лише навчальний / прототипний UI.

## Що є

- Шапка: телефон, години, wishlist, пошук, кошик  
- Горизонтальне меню категорій  
- Сайдбар: фільтри (бренд, тип, матриця, ціна)  
- Сітка товарів: бейджі (знижка / хіт / популярний), рейтинг, ціна, «Купити»  
- Сортування, ліміт на сторінку, пагінація  
- Блоки брендів і відгуків  
- Кошик (localStorage) + wishlist  
- Адаптивна верстка  

## Запуск

Відкрийте `index.html` у браузері або підніміть локальний сервер:

```bash
# Python
python -m http.server 8080

# Node
npx serve .
```

Потім: http://localhost:8080
